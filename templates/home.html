Project Purpose

Create an application that allows writers to organize their ideas and thoughts in one place while giving them the ability
to interact and collaborate with other users.

Timeline

Week 1

Create a working home page and login/create account page.
Allow users to create accounts and login with these accounts.
Create a text editor interface where users can write.
Have functioning backend so website can be visited through localhost.

Week 2

Improve text interface to a more notebook style.
Allow each notebook to have multiple tab and users to have multiple notebooks (similar to OneNote).
Create editing functionality (similar to editing mode on google docs).
Enable project sharing between different users.

Week 3

Give users certain access levels (editing/viewing) for specific projects.

Week 4

Improve website UI and make any final touches.
Create public portal where users can submit for anyone to see.
Add additional features (plot timeline planning, etc).
